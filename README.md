# VendasMVC

Projeto em C# utilizando o padrão MVC(Model-View-Controller). Consiste em uma aplicação Web
de um sistema de vendas com uma interface gráfica para manipular o banco. 

## Tecnologias utilizadas

- C#
- .NET
- ASP.NET Core MVC
- Entity Framework
- LINQ
- MySQL
- Migrations
- HTML
- CSS
- Bootstrap
